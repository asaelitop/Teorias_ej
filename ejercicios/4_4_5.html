Ejemplo 4.4.5

Modificar Elemento (Arreglo Desordenado)

Escribe un programa que pida al usuario una lista de números, un valor a buscar dentro de la lista y un nuevo valor para reemplazarlo.

Requisitos

Todo el código debe estar dentro del método main.

  • Pide el tamaño del arreglo con: "Ingrese el tamaño del arreglo (N): ".
  • Pide los elementos del arreglo con: "Ingrese los [n] elementos del arreglo:".
  • Pide el elemento a modificar con: "Ingrese el elemento a modificar (X): ".
  • Pide el nuevo valor con: "Ingrese el nuevo valor (Y): ".
  • Busca la primera aparición del elemento a modificar y reemplázalo por el nuevo valor.
  • Si la modificación es exitosa, muestra: "Arreglo modificado:" seguido del nuevo arreglo.
  • Si el elemento a modificar no se encuentra, muestra: "El elemento [X] no está en el arreglo.".
  • Si el tamaño inicial del arreglo es 0, muestra: "El arreglo está vacío.".

Ejemplo de uso esperado

Ingrese el tamaño del arreglo (N): 5
Ingrese los 5 elementos del arreglo:
10
20
30
40
50
Ingrese el elemento a modificar (X): 30
Ingrese el nuevo valor (Y): 99
Arreglo modificado:
10 20 99 40 50 

Ingrese el tamaño del arreglo (N): 3
Ingrese los 3 elementos del arreglo:
1
2
3
Ingrese el elemento a modificar (X): 99
Ingrese el nuevo valor (Y): 100
El elemento 99 no está en el arreglo.

Concepto Clave: Banderas (Flags)

Una "bandera" (o flag) es una variable, usualmente de tipo booleano (true/false), que usas para señalar que un evento ha ocurrido. Es muy útil para controlar el flujo de tu programa.

Tip: Usando una Bandera

Crea una variable boolean encontrado = false;. Cuando encuentres el número que buscas, cámbiala a true.

Puedes usar esta bandera para dos cosas: 1) para detener el bucle de búsqueda una vez que ya encontraste el elemento, y 2) para decidir al final si imprimes el arreglo modificado o el mensaje de "no encontrado".
