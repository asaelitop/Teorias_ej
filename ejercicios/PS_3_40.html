Problema Suplementario 3.40

Análisis de Conductividad de Elementos

Escribe un programa que lea el nombre, la conductividad eléctrica y la conductividad térmica de varios elementos. El programa debe detenerse cuando el usuario ingrese "NN" como nombre. Al final, debe reportar cuál es el mejor y peor conductor eléctrico, y el mejor y peor conductor térmico.

Requisitos

Todo el código debe estar dentro del método main.

  • Usa un bucle do-while para pedir datos hasta que el usuario ingrese "NN".
  • Dentro del bucle, pide el nombre, la conductividad eléctrica y la conductividad térmica.
  • Guarda estos datos en tres arreglos paralelos (uno para nombres, uno para cond. eléctrica y otro para cond. térmica).
  • Después de que el usuario termine de ingresar datos, usa bucles for para encontrar los elementos con los valores máximos y mínimos para cada tipo de conductividad.
  • Imprime un reporte final con los resultados.

Ejemplo de uso esperado

--- Ingrese los datos de los elementos ---
Ingrese el nombre del elemento (o 'NN' para finalizar): Plata
Ingrese la conductividad electrica: 63.0
Ingrese la conductividad termica: 429.0
Ingrese el nombre del elemento (o 'NN' para finalizar): Azufre
Ingrese la conductividad electrica: 0.1
Ingrese la conductividad termica: 0.2
Ingrese el nombre del elemento (o 'NN' para finalizar): NN

--- Resultados del Análisis de Conductividad ---
Mejor conductor eléctrico: Plata
Peor conductor eléctrico: Azufre
Mejor conductor térmico: Plata
Peor conductor térmico: Azufre

Concepto Clave: Arreglos Paralelos

Los arreglos paralelos son una técnica donde usas múltiples arreglos y el índice `i` se refiere a la misma entidad en todos ellos. Por ejemplo, `nombres[i]`, `condElectrica[i]` y `condTermica[i]` contienen toda la información del i-ésimo elemento.

Tip: Encontrando el Máximo/Mínimo

Para encontrar el valor máximo en un arreglo, inicializa una variable `max` con el primer valor del arreglo. Luego, recorre el resto del arreglo con un bucle `for`. Si encuentras un valor mayor que `max`, actualiza `max` con ese nuevo valor. Haz lo mismo para el mínimo.
