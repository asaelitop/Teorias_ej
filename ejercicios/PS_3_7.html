Problema Suplementario 3.7

Calculadora de Nómina con Aumentos por Rango

Una empresa desea calcular su nómina total después de aplicar aumentos de sueldo. Los aumentos se basan en el sueldo actual del empleado: 10% si ganan menos de $10,000; 7% si ganan entre $10,000 y $25,000; y 8% si ganan más de $25,000.

Requisitos

Todo el código debe estar dentro del método main.

  • Pide al usuario el número de empleados con: "Introduce el número de empleados: ".
  • Usa un bucle for para procesar cada empleado.
  • Dentro del bucle, pide el sueldo de cada uno con: "Sueldo del empleado #[num]: ".
  • Usa una estructura if-else if-else para calcular el nuevo sueldo según el rango.
  • Imprime el nuevo sueldo de ese empleado, formateado a dos decimales: " -> Nuevo sueldo: $[sueldo]".
  • Mantén un acumulador para sumar todos los nuevos sueldos.
  • Al final del bucle, imprime la nómina total con: "El total de la nueva nómina es: $[total]".

Ejemplo de uso esperado

Introduce el número de empleados: 3
Sueldo del empleado #1: 5000
 -> Nuevo sueldo: $5500.00
Sueldo del empleado #2: 20000
 -> Nuevo sueldo: $21400.00
Sueldo del empleado #3: 30000
 -> Nuevo sueldo: $32400.00

El total de la nueva nómina es: $59300.00

Rangos y Condiciones

Presta atención a los límites de los rangos. El rango intermedio incluye el 10,000 y el 25,000 (sueldo <= 25000).

Tip: Acumuladores

Un "acumulador" es una variable que usas para llevar una suma continua. Declara una variable double totalNomina = 0; antes del bucle.

Dentro del bucle, después de calcular cada nuevoSueldo, simplemente añádelo al total con totalNomina += nuevoSueldo;.
